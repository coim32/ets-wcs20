Compliance Test Suite for OGC Web Coverage Service 2.0 - XML/POST Protocol Binding Extension

Overview

This executable test suite is based on the following OGC specifications:

  • OGC Web Coverage Service 2.0 Interface Standard - XML/POST Protocol Binding Extension (1.0) (OGC 09-148r1)
  • OGC GML Application Schema - Coverages (09-146r2)
  • Compliance Test Language (CTL) 0.9

What is tested

The following operations of the WCS 2.0 XML/POST extension are tested.

  • GetCapabilities (KVP)
  • DescribeCoverage (KVP)
  • GetCoverage (KVP) on GML coverage

Test data

Sample coverages can be found at http://schemas.opengis.net/wcs/2.0/examples.
